# BigMart Sales Dashboard - Power BI

## Overview

This project presents an interactive Power BI dashboard developed using the BigMart Sales Dataset. The dashboard transforms raw retail sales data into meaningful visual insights that support business decision-making.

## Objectives

* Analyze sales performance across outlet types and locations.
* Identify customer purchasing patterns.
* Evaluate product category performance.
* Build an interactive dashboard for data-driven insights.

## Tools Used

* Power BI
* Power Query
* DAX
* Data Visualization

## Dashboard Features

* Total Sales KPI
* Average Sales KPI
* Item Type Analysis
* Outlet Performance Analysis
* Sales Trend Analysis
* Interactive Filters and Slicers

## Key Insights

* Certain outlet types contribute significantly more revenue.
* Customer preferences vary across product categories.
* Sales performance differs across location tiers.
* Interactive dashboards help uncover actionable business insights.
